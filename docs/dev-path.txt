To start a new project:
    1. MaKe DIRectory for example "projects", CD to it.
    2. MKDIR "urprojectname", cd to it.
    3. Create virtual environment with python3: "virtualenv . --python=python3"
    4. Activate it: source bin/activate
    5. Install Django: pip install django
    6. Start new project: django-admin.py startproject yourproject
    7. Now you can start server: python manage.py runserver
